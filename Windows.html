AFS to Windows

Tutorial:

  1. Click here to download the tool.


  2. Move the tool into the folder that you’d like your files to be downloaded to.


  3. Double click the file and wait for a window to appear. You may get a security warning popup, in which case click “Run” to certify that you wish to run the program. Don’t worry, it’s completely safe.


  4. Enter your uniqname when it prompts you to.


  5. Enter your password when it prompts you to. Note that the cursor will not move as you type.


  6. The file transfer will begin and a list of the files being downloaded will scroll across the screen. Note that this may take a while depending on the size of your files and the speed of your internet connection.


  7. A message will appear letting you know when the transfer is finished. Press any key to exit the window.


  8. You will find all of your files in a folder labeled “Files from AFS” located in the same location as the tool you downloaded.


This tool makes use of PuTTY to securely transfer files from AFS to a local Windows computer using the SFTP protocol. It is compatible with any computer running Windows XP and up. This application downloads the AFS directory to the directory in which the application is run. Note for power users: this tool will not copy over any dotfiles from the top level directory.
